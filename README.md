## Overview

KEX Mod Menu delivers a web-first mod menu layout with a cyberpunk-flavored presentation and a bright rainbow animated backdrop. The interface is built to make large sets of options easier to browse, using clear grouping, live search, and compact controls that work well on both desktop and mobile screens.

It is a solid fit when you need a refined browser UI for sorting through menu items and switching them on or off quickly. The experience also includes keyboard shortcuts and context menu actions, so navigation stays efficient while a built-in counter keeps the number of active mods easy to see.

## Usage

Open the interface in your browser and use the category panel to jump between option groups.

Typical workflow:
1. Search for a mod or feature name.
2. Use the toggle controls to enable or disable entries.
3. Open the context menu for item-specific actions.
4. Use keyboard shortcuts when you want faster navigation.
5. Check the counter to see how many mods are currently active.
